# Value Narratives — Quick Reference

> Comprehensive training: [Sales Track — Value Narrative Selection](../../courses/tracks/sales/03-value-narrative-selection.md) | [Sales Track — Operating Model Transformation](../../courses/tracks/sales/01-operating-model-transformation.md)

## Narrative Selection Matrix

| Buyer Persona | Lead With | Key Metric | What They Need to Hear |
|---------------|----------|------------|----------------------|
| **CTO / VP Eng** | Capacity Unlocking | Story points reclaimed from maintenance | "Engineers review PRs (minutes) instead of implementing (hours). Backlog of deferred work actually gets done." |
| **CISO / Security** | Risk Reduction | MTTR for critical findings | "Exposure window shrinks from 'next sprint' to 'next CI run'. Findings remediated as detected." |
| **FinOps / CFO** | Cost Optimization | ACU cost vs. equivalent engineer-hours | "Consumption-based, directly attributable ROI. Licensing cost avoidance accelerates migration payback." |
| **Delivery Director** | Velocity Multiplication | Timeline compression ratio | "Migration drops from months to weeks — agents parallelize, engineers review PRs." |
| **Practice Lead / P&L** | Operating Model Efficiency | Cost per unit of work | "Most competitive cost structure in RFP scenarios. Margin expands as playbooks mature." |

## The Seven Narratives

| Narrative | Audience | Core Argument |
|-----------|----------|---------------|
| **Capacity Unlocking** | Eng Management | Devin executes deferred "should-do" work autonomously — dep updates, test gaps, docs debt |
| **Engineering Focus Elevation** | Eng Leadership | Shift from 60% implementation to 30% implementation + 30% design + 30% review |
| **Velocity Multiplication** | Delivery, PM | N child agents in parallel — 50 services upgraded in days, not weeks |
| **Quality Improvement** | QA, Security | Consistent automated checks on every PR, scheduled scans, systematic test generation |
| **Risk Reduction** | CISO, CTO | MTTR drops, vulnerability windows shrink, knowledge encoded in playbooks (bus factor decreases) |
| **Cost Optimization** | FinOps, Finance | ACU consumption-based pricing, predictable budgets, licensing cost avoidance via migration |
| **Operating Model Efficiency** | SI Practice Leads | Hybrid human + agent model — efficiency first, quality rises, outcomes improve, margin expands |

## Pain Point → Narrative Quick Lookup

| Pain Point | Lead With |
|-----------|----------|
| "Can't hire fast enough" | Capacity Unlocking |
| "Security posture is reactive" | Risk Reduction |
| "Migrations take too long" | Velocity Multiplication |
| "Losing RFPs on price" | Operating Model Efficiency |
| "Engineers doing grunt work" | Engineering Focus Elevation |
| "Can't predict project costs" | Unit-of-Work Economics (see [Sales Track](../../courses/tracks/sales/02-unit-of-work-economics.md)) |

## Key Rules
- Match the narrative to the buyer persona — same capability, different framing
- Lead with the pain point, not the technology
- Frame Devin as team augmentation, not replacement
- Prepare persona-specific evidence (metrics and examples) before every conversation
