# Workshops by Technical Role

Hands-on workshops organized by job function. Each workshop bundles modules into a structured sequence covering the engineering concerns most relevant to that role.

**Prerequisite:** [Foundations](../foundations/) (Level 1) and the relevant [Training Track](../tracks/) (Level 2)

## Available Workshops

| Workshop | Focus | Duration |
|----------|-------|----------|
| [Application Development & Maintenance](development/app-maintenance/) | Feature dev, bug fixing, code maintenance — full app lifecycle | 4-6 hours |
| [Feature Development](development/feature-development/) | New features on existing applications | 1-2 hours |
| [Digital Engineering](digital-engineering/) | CI/CD, cloud infrastructure, observability, platform engineering | 4-6 hours |
| [Quality Engineering & Assurance](quality-engineering/engineering/) | Test automation, E2E testing, continuous quality, code review | 4-6 hours |
| [Quality Engineering & Security](quality-engineering/engineering-security/) | QE + security vulnerability remediation combined | ~3 hours |
