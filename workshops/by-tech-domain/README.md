# Workshops by Technical Domain

Deep-dive workshops organized by technology area. Each workshop bundles modules into a structured sequence targeting a specific engineering challenge.

**Prerequisite:** [Foundations](../../courses/foundations/) (Level 1) and the relevant [Track](../../courses/tracks/) (Level 2)

## Available Workshops

| Workshop | Focus | Duration |
|----------|-------|----------|
| [COBOL Modernization](modernization/cobol/) | End-to-end mainframe modernization — system understanding through execution | ~4 hours |

| [.NET Cloud-Native Modernization](modernization/dotnet-cloud-native/) | .NET monolith → Kubernetes-hosted cloud-native APIs | 2.5-3 hours |
| [Framework Upgrades](modernization/framework-upgrades/) | Angular + Spring Boot version upgrades at scale | 1-2 hours |
| [Data Source Migration](modernization/data-source-migration/) | Legacy data warehouse → modern schema + app reconnection | 1-2 hours |
| [Platform Microservice Decomposition](modernization/platform-decomposition/) | Monolith → platform-conformant microservices with IaC | 1.5-2 hours |
