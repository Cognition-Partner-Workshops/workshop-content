# Evaluation: D3 — Full Automation Pipeline (Capstone)

## Blocking Criteria

- [ ] Participant designed or implemented at least one automation layer (event-driven, scheduled, or on-demand)
- [ ] Participant can explain the three-layer automation strategy
- [ ] Participant can articulate the Ask → Session → Playbook → Automation progression

## Recommended Criteria

- [ ] At least two layers were designed with specific triggers, prompts, and expected outputs
- [ ] The issue-to-Devin workflow was implemented or designed with bot-loop prevention
- [ ] A Playbook was created for on-demand use (e.g., service scaffolding)
- [ ] Participant can draw the complete automation architecture diagram
- [ ] Participant produced an automation strategy summary document
- [ ] Participant discussed how the shared context layer ties all three layers together
