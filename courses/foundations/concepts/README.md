## Reading Order

Read in order — each section builds on the previous:

1. **Cloud vs. Local** establishes the two execution paradigms
2. **Event-Driven Reactions** shows why automation beats human paging
3. **Agent Orchestration** scales the model to teams and workloads
4. **What to Give AI** gives you the decision framework for daily use
